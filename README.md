# FlowFence-Lite

This repository is the working topic workspace for `FlowFence-Lite`.

It was created from the shared `_template/` and is intended to stay independent and self-contained. Use this folder for all contract writing, baseline planning, experiments, results, and paper-facing artifacts for this topic only.

## Execution Order

1. Fill the contract in `research/contract/`.
2. Plan and reproduce baselines.
3. Only then implement and evaluate the proposed method.
4. Draft claims only from saved evidence in `results/`.

## Immediate Startup

1. Fill the five numbered contract files in `research/contract/`.
2. Update `research/logs/roadmap.md` with the current owner and first decision.
3. Use `research/logs/progress.md` to log initialization and future work.
4. Do not implement research code until the contract and baseline plan are written.

## Topic Scope

`FlowFence-Lite` should remain a single-topic repo. Do not mix unrelated privacy-defense ideas into this folder.
